This is the modified version of Astrid starting from the astrid model downloaded from [[github](<https://github.com/saravanan-thirumuruganathan/astrid-string-selectivity>)]

## Script

```bash
# quick test
./run.sh test 

# exp on DBLP 
./run.sh dblp

# all exp
./run.sh all
```
